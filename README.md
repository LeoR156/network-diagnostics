# Простой BASH скрипт для диагностики сети.

Сохраняет отчёт о диагностике в файл `"network_report_${DATE}.txt"`

### Что именно проверяет и сохраняет:
* `ping` на 8.8.8.8 (доступность интернета).
* Проверяет DNS-резолвинг (перевод имён в IP-адреса).
* Внешний IP-адрес.
* Таблицу маршрутизации текущего хоста.
* Список открытых портов.
* Доступность google.com на портах 80 и 443.

### Инструкция по использованию:
* Клонируйте репозиторий командой: `git clone https://github.com/LeoR156/network-diagnostics.git`
* Перейдите в папку со скриптом: `cd network-diagnostics`
* Проверьте права файла командой `ls -la network_check.sh`. Если отсутствует флаг `x`, сделайте скрипт исполняемым командой: `chmod +x network_check.sh`
* Запустите файл командой `./network_check.sh`. Подождите несколько секунд, пока отчет создастся. После успешного выполнения вы получите соответствующее сообщение в консоли.
* Прочитать отчет можно командой (вместо звездочки можно вписать точную дату): `cat network_report_*.txt`

### Как автоматизировать запуск (cron):
* Выполните действия из инструкции выше.
* Откройте файл *crontab* командой `sudo crontab -e`
* Добавьте в конец файла с новой строки следующую конфигурацию: 
  `*/30 * * * * /полный/путь/к/папке/network_check.sh`
* Скрипт будет выполняться **каждые 30 минут**. Если вы хотите задать другое расписание, воспользуйтесь удобным тренажером: [crontab.guru](https://crontab.guru/)
